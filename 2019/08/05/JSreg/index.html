Javascript用正则表达式替代字符串中的标点符号

发表于 2019-08-05 | 分类于 coding

当在处理字符串的时候要对其中的标点符号进行处理时，用if==&&肯定是行不通的，因为太麻烦了，这时候正则表达式就派上用场了
正则表达式以/开始，以/结尾
[]表示匹配其中所有内容
记住标点符号要用转义符号，之间用|间隔
/[\-|\_|\(|\)|\+|\=|\{|\[|\]|\}|\;|\:|\'|\"|\,|\.|\/|\?]/g
g代表全局模式，运用于所有字符串，即匹配所有符合条件的字符
