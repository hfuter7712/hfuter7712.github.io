Vue.js学习（一）

发表于 2019-10-04 | 分类于 Coding(Front End)

来自菜鸟教程
https://www.runoob.com/vue2/vue-tutorial.html
是一套构建用户界面的渐进式框架

CDN如下
https://cdn.staticfile.org/vue/2.2.2/vue.min.js

首先需要通过实例化Vue 

var vm=new Vue({
   //选项  
})

el
Vue构造器中的一个参数，是DOM元素中的id
example:
el: '#my_id'
data
用于定义属性 

data:{
     site:"google",
     url"www.google.com"
},

method
用于定义函数，可用return来返回函数值 

method:{
    detail:function(){
         return this.site+"添加文本内容";
          }
       }

双大括号
用于输出对象属性和函数返回值 

<h1>{{site}}</h1> 
<h1>{{details()}}</h1>

当data中引用了部分外界的原始数据，当改变Vue实例中这些对应的数据时，原始数据也会跟着一起改动，同理，当原始数据改动时，Vue实例中的数据也会一起改变

指令
v-html
可用于输出html代码 

<div v-html="test"></div>

在Vue对象中的data：

data:{
    test:'<h1>夜夜夜</h1>'  
}

将会展示一个一级标题夜夜夜如下图

夜夜夜

v-bind
用于HTML属性中的值
<a v-bind:href="url">
就将该链接改成了Vue实例中url data所对应的的url

v-if
如果所绑定的值为true,则显示该元素，否则不显示
<p v-if="seen">测试</p>
如果seen为true,则会显示文本“测试”

v-else
跟在v-if后面，当if和else-if都不满足条件时执行该else

v-else-if
跟在if后面，其实上面三条语句跟C语言中的if,else if和else语法相同
example： 

<div id="app">
    <div v-if="type === 'A'">
      A
    </div>
    <div v-else-if="type === 'B'">
      B
    </div>
    <div v-else-if="type === 'C'">
      C
    </div>
    <div v-else>
      Not A/B/C
    </div>
</div>

v-on
用于监听DOM事件
<a v-on:click="thingFirst">

v-model
可以在指定的表单控件元素上创建双向数据绑定，也就是说，通过榜单中输入值，借助这个指令可以更新Vue实例中data的值
<input v-model="message">

指令缩写
v-bind
<a v-bind:href="url"></a>
缩写:
<a :href="url"></a>

v-on
<a v-on:click="thingA></a>
缩写
<a @click="thingA></a>

循环语句
v-for
可以让data中的数组循环渲染 

<ol>
<li v-for="site in sites">
    {{site.name}}  

</li>
</ol>

在Vue实例中的data中的sites  

sites:[
     {name:'first'},
     {name:'second'},
     {name:'third'}
]

渲染出来的效果就是一个有序列表，分别对应着  
1.first
2.second
3.third

个人感觉很适合大量数据的列表显示
还可以循环整数 

<ul>
   <li v-for="n in 10">
   {{n}}  

   </li>
</ul>

计算属性computed 

computed:{
 reverseMessage:function(){
          return this.message.split('').reverse().join('')
   }
}

与method的区别（来自菜鸟教程）:效果是一样的，但是computed是基于它的依赖缓存，只有相关依赖发生改变的时候才重新取值，而methods在重新渲染的时候函数会重新调用执行

监听属性watch
可以通过watch来响应数据变化 

vm.$watch('v-model所绑定的变量',function(before,after){
   console.log("之前的值为:"+before+" 之后的值为:"+after);  
});

样式绑定
<div v-bind:class="{active: isActive}"></div>
在Vue实例data中定义isActive为true或者false可以将该div的类设置为active或者解除设定，若isActive为true，则相当于如下
<div class="active"></div>

表单
复选框
<input v-model="checkbox" v-model="checked"> 

<input type="checkbox" id="google" value="google" v-model="selectNames">  
<input type="checkbox" id="baidu" value="baidu" v-model="selectNames">  
<input type="checkbox" id="yahoo" value="yahoo" v-model="selectNames">

在Vue实例中定义data selectNames:[],然后在前面表单后方键入如下代码
<p></p>
即可看到其中显示自己已经选中的值
比如选中google与百度
显示如下
[google,baidu]

Vue最强大功能:组件
组件可以扩展HTML元素，封装可以重复使用的代码
注册一个全局组件语法格式如下:
Vue.component(tagName,option)
注册完成后可以用以下方式来调用组件
<tagName></tagName>

局部组件 

<script>
var Target={
  template:'<h1>模板组件1</h1>  
};
new Vue({
  el:'#app',
  components:{
   'test':Target
}
});

这样<test></test>组件就只能在id为app的Vue实例里使用了

自定义指令
注册一个全局自定义指令 

Vue.directive('focus,{
 inserted:function(el){
 el.focus()
}
})

这样当页面载入时，标签上有v-focus的元素自动获得焦点
